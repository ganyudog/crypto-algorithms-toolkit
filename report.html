AES

AES 支持 128/192/256 位密钥，实现 CBC/ECB/CFB/OFB/CTR 五种工作模式，使用 PKCS7 填充。

SM4

SM4 分组长度 128 位，密钥长度 128 位，支持 ECB 和 CBC 模式。

RC6

RC6 参数为 w=32、r=20，分组长度 128 位，支持可变长度密钥，参考 RFC 2040 实现。

RSA

RSA 使用 2048 位密钥，OAEP 填充 (SHA1)，PKCS1v15 签名。

ECC/ECDSA

ECC 使用 SECP192R1 曲线，支持 ECDH 密钥交换和 ECDSA 数字签名。

五、API 接口

基于 Flask 提供 RESTful API。

端点方法说明
/api/healthGET健康检查
/api/algorithmsGET所有算法和参数
/api/{algorithm}/{action}POST/GET通用密码算法接口

调用示例

# AES 加密
curl -X POST http://127.0.0.1:5000/api/aes/encrypt \
  -H "Content-Type: application/json" \
  -d '{"data": "Hello World", "mode": "CBC"}'

# SHA256 哈希
curl -X POST http://127.0.0.1:5000/api/sha256/hash \
  -H "Content-Type: application/json" \
  -d '{"data": "Hello World"}'

# RSA 密钥生成
curl -X POST http://127.0.0.1:5000/api/rsa/generate \
  -H "Content-Type: application/json" \
  -d '{"key_size": 1024}'
